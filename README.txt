== Description ==

"Easy Videos" with the following features:
⦁	As an admin, I should be able to bulk import videos from the Linus Tech Tips youtube channel (https://www.youtube.com/c/LinusTechTips/videos) to a custom post type video using youtube API (https://developers.google.com/youtube/v3/docs)
⦁	Upon importing, the following information from the YouTube API be saved as a meta field: title, description, medium thumbnail url, publish time.
⦁	Able to select which videos you want to import and also have its preview before importing them.
⦁	If any channel or user has a lot of videos, there will be pagination for them as well.
⦁	Videos playable on frontend via shortcode [easyvideos].



== Installation ==

Requirement:
- WordPress 6.1
- PHP Version 7.4.19


1. Upload `easy-videos.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
