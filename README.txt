== Description ==

"Easy Videos" with the following features:
⦁	Easy video is able to bulk import videos from youtube channel, able to select which videos you want to import and also have its preview before importing them.
⦁	If any channel or user has a lot of videos, there will be pagination for them as well.
⦁	Videos playable on frontend via shortcode [easyvideos].



== Installation ==

Requirement:
- WordPress 6.1
- PHP Version 7.4.19


1. Upload `easy-videos.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
